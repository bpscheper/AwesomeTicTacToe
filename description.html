Description

Awesome Tic Tac Toe is a version of Tic Tac Toe that uses bigger playing boards and a different scoring.
In Awesome Tic Tac Toe the user gets to chose what siz board they would like to play on between the sizes of a 4X4 and 10X10 board.
Once the sizes are choses then the game will begin. The Scoring for this game will be calculated based on how many blocks can you get in a row, column, and diagonal.
The user with the highest amount of connecting blocks wins!


I hope you enjoy this Awesome Tic Tac Toe Game!
